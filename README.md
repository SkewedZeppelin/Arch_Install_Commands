Arch Install Commands
=====================

Instructions
------------
Run the following commands to be greeted with a wholesome install within minutes
```shell
sh Pre-Install.sh
sh Post-Install.sh #Midway run 'sh Arch_Linux-Package_Installer.sh'
```
